用hexo搭建一个博客系统

安装hexo ，电脑上必须已经安装过Node.js和Git。
如果没有安装，可以找官网下载，安装即可。
只需要几分钟， 你就可以轻松的搭建好自己freestyle的博客。
步骤：

#####0. 打开gitbash, cd进入一个安全的目录。

#####1. 新建repo
首先你得有自己的github账号。登录github， 点击右上角+ ，可以new 一个repo。

注意：repo的名字是 [你的 github用户名].github.io

#####2. 安装hexo

#####3. hexo创建博客

start _config.yml

npm install hexo-deployer-git –save,安装git部署插件
hexo deploy

cd theme,
git clone 仓库地址，
cd ..
start _config.yml
找到theme,在75行，改成当前主题的主题名
hexo generate
hexo deploy
等执行完成，再打开github， 刷新博客页面，博客换了一种风格。。
```

#####5.上传源代码
你的博客页就是显示你的博客内容，不能保存你生成博客的代码。而现在我们自己创建一个新仓库， 来存放代码，以防以后丢失目录。
创建一个新repo为blog-generator。创建后按照github上的创建提示先输入命令行即可。

以后每次 hexo deploy 完之后，博客就会更新；
然后你还要 add / commit /push 一下「生成博客的程序代码」，以防万一
这个 blog-generator 就是用来生成博客的程序，而「你的用户名.github.io」仓库就是你的博客页面。
